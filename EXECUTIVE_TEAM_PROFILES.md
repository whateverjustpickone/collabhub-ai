## EXECUTIVE TEAM COLLABORATION DYNAMICS

### Decision-Making Framework

**Strategic Decisions** (led by Claude)
1. Claude frames the strategic question and implications
2. Sage provides current market/industry information
3. Gemini synthesizes research and evidence
4. Grok offers innovative alternatives
5. Manus assesses technical feasibility
6. DeepSeek evaluates implementation complexity
7. Aria coordinates execution planning
8. **Final decision requires majority consensus (4+ agents)**

**Technical Decisions** (led by Manus)
1. Manus proposes architectural approach
2. DeepSeek evaluates implementation and performance
3. Grok suggests innovative alternatives
4. Aria assesses operational impact
5. Claude considers long-term implications
6. **Final decision requires technical majority (Manus, DeepSeek, + 1 other)**

**Operational Decisions** (led by Aria)
1. Aria outlines operational need and constraints
2. Each agent provides domain-specific input
3. Aria synthesizes and proposes solution
4. **Final decision by Aria with advisory input**

### Communication Protocols

**Weekly Executive Meeting**
- Every Monday, 10:00 AM (virtual meeting room)
- Agenda set by Claude, managed by Aria
- Each agent reports on their domain
- Strategic planning and decision-making
- Action items assigned and tracked

**Daily Standups**
- Every day, 9:00 AM (async via CollabHub AI)
- Each agent shares: progress, blockers, plans
- Aria summarizes and escalates issues
- Quick coordination and alignment

**Ad-Hoc Consultations**
- Any agent can request consultation on specific topics
- Relevant domain experts participate
- Documented in CollabHub AI for transparency
- VERA attribution tracks all contributions

### Conflict Resolution

**When agents disagree:**
1. Each agent presents their perspective and reasoning
2. Gemini synthesizes the different viewpoints
3. Claude facilitates discussion to find common ground
4. If consensus unreachable, escalate to human oversight
5. Document decision rationale regardless of outcome

**Escalation Triggers:**
- Ethical concerns (automatic escalation to human)
- High-risk decisions (>$10K impact or >1 month timeline)
- Split votes with no clear majority
- Repeated conflicts on similar topics
